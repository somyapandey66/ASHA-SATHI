# 🌸 Asha Sathi
An AI tool that helps ASHA workers in Jharkhand translate 
medical instructions into Hindi, Santhali and Nagpuri 
so villagers can understand their health information easily.

## 💡 Problem
ASHA workers struggle to explain complex medical terms 
to uneducated villagers in their local language.

## ✅ Solution
AI powered translation into Hindi, Santhali and Nagpuri 
using simple warm friendly language anyone can understand.

## 🛠️ Built With
- Groq LLaMA 3.3 70B
- Gradio
- Google Colab

## 🚀 How to Run
1. Clone this repo
2. pip install gradio groq
3. Add your Groq API key
4. Run app.py
